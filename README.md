# Hospital Management System (C++)

## Overview

The Hospital Management System is a console-based application developed in C++ as an academic project. It demonstrates Object-Oriented Programming (OOP) concepts and Data Structures by managing patient records, doctor information, and hospital bed allocation.

## Features

- Add Patient Records
- Display Patient Records
- Delete Patient Records
- Manage Doctor Information
- Hospital Bed Allocation
- Menu-Driven Interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Linked Lists
- Dynamic Memory Allocation
- Pointers
